== Description ==

Plugin that add a share to friends jQuery Lightbox. Users will be able to share your content by 3 ways. Sending email manually, using Facebook or Twitter, using Open Inviter to get contacts from your email provider.


== Installation ==

1. Upload and activate the plugin
2. Go to the Recommend a friend option page
* Complete your Open Inviter login if you want to user the feature
* Choose the feature you want to enable
* You can autoadd the button after you posts content by checking the box
 
3. Add the Recommend a Friend widget or use the php function  `<?php echo recommend_a_friend_link( $permalink, $image_url, $text_link ); ?>`
* $premalink is the URL you want to be shared
* $image_url (facultative) the image url used instead of the default one
* $text_link(facultative) the text you want to display instead the image (you need to choose between display an image or a text)
